# ITDA Language 개발계획서

**프로젝트명:** 잇다 (ITDA Language)  
**목적:** 농인(청각장애인)이 텍스트를 입력하면 자연스러운 음성으로 출력하여 청인과 소통할 수 있도록 지원하는 앱  
**태그라인:** "소리가 없어도, 우리는 이어집니다"  
**작성일:** 2026-06-15  
**버전:** 0.1.0

---

## 1. 프로젝트 개요

### 1.1 배경
청각장애인이 일상 대화에서 구두언어로 소통하는 데 어려움이 있다. 텍스트를 입력하면 스피커폰으로 음성을 출력하여 상대방이 들을 수 있도록 하는 앱이 필요하다.

### 1.2 핵심 가치
- 농인이 **즉시** 음성으로 소통할 수 있는 실시간 TTS
- 사용자별 **맞춤 음성** (남녀, 연령, 톤 선택)
- **시각적 피드백** (텍스트 하이라이트, 이퀄라이저)으로 음성 출력 상태 확인
- **다국어 지원** (한국어, 영어, 중국어, 일본어)

### 1.3 기술 스택
| 구분 | 기술 |
|---|---|
| Android | Kotlin + Jetpack Compose, Hilt DI, Retrofit |
| Backend | Python FastAPI, SQLAlchemy, Edge TTS / CLOVA Voice |
| DB | SQLite(개발) / PostgreSQL(프로덕션) |
| 캐시 | Redis |
| 배포 | Docker + Nginx + Let's Encrypt |
| 인프라 | Naver Cloud Platform |

## 9. 배포 계획

### 9.1 서버 배포 (Naver Cloud Platform)
```bash
# 1. 서버 접속
ssh user@api.itda.kr

# 2. 배포 실행
cd /opt/itda && ./deploy/deploy.sh

# Docker 구성: FastAPI + Celery + PostgreSQL + Redis + Nginx + Certbot
```

### 9.2 Android 배포 (Google Play Store)
```bash
# 1. 키스토어 생성
keytool -genkey -v -keystore itda-release.jks -alias itda \
  -keyalg RSA -keysize 2048 -validity 10000

# 2. gradle.properties에 키스토어 정보 입력

# 3. Release APK/AAB 빌드
./gradlew bundleRelease

# 4. Google Play Console 업로드
```
